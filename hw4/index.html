Functionality

The main function of my dashboard is to portray various statistics relating to budget of different movies. The first plot displays the average budget spent on movies for a given year. This is conveyed in an area line chart. I have chosen area over a strict line in order to emphasize the overall trend for movies to have higher budget in more recent years. Although area may increase the ile factor slightly, I believe that since only one statistic is being conveyed, the graph would seem very sparse with only a line portrayal.

The second graph is a bar chart of average budget by genre. While the previous graph showed a discrepancy between year and budget, this graph shows a discrepancy between genre and budget. A bar chart has a very easy interpretation for the user and limited lie factor.

Lastly, the scatterplot converys information of average budget and average rating aggregated by genre. Each data point represents a genre and average rating and average budget is plotted. This gives the user an idea of how budget and rating may be connected. My first attempt at visualizing this did not involve aggregating by genre and resulted in a very dense graph.This was hard to interpret because the majority of points were plotted on top of one another. This led to the need of further aggregation, achieved by grouping by genre. While the lie factor is increased using this technique, it is a trade off which makes the concept of rating versus budget more easily interpreted by users.

The colors were chosen according to colorbrewer for unordered, categorical data. Since the three elements of the dashboard are only connected in underlying relation to budget, I wanted colors to imply dissimilarity between information being visualized.

Data preprocessing

Data preprocessing was essential in order to visualize this data. This was performed in R. Since my dashboard deals with each movie's budget statistic, the first step was to remove every row for which budget did not exist. Since I am further aggregating by genre, I have chosen to select only rows in which a movie is classified as being only one genre. This made aggregation into disjoint genres achievable. I have also performed various binning techniques to display histograms of the number of movies within certain budget ranges. While I did not end up displaying this visualization, code for creation (in R) of the dataframe with this information can be found in the data folder.
