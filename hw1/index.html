Homework 1

Assignment Description: Create an interactive bar chart displaying frequencies of letters in a user entered string.

Sort

Graph Attributes

  • Modify bar colors: Bars are shaded on a gradient corresponding to frequency of letter, with more frequent letters appearing with darker bars (C-level)
  • Modify tick label formatting: Changed number of decial points visible to 0 (C-level)
  • Modify tick label formatting: Changed frequency of y-axis labels based on max letter count (C-level)
  • Modify grid and axis lines: Inserted y-axis grid lines based on number of y-axis tick marks (C-level)
  • Modify grid and axis lines: Made y-axis lines dashed and lighter grey (C-level)
  • Added button allowing user to sort bars alphabetically or by decreasing letter count (B-level)
  • Added custom tooltip showing exact value of bar on mouseover (B-level)
  • Added search box for users to search rather than using the URL query (B-level)
  • Search box maintains user input (even across page refresh) for easy alteration of string input (A-level)
  • At start of visualization, bars grow from bottom up and bounce into place (A-level)
  • On first mouseover, bars change to black permanently
  • On subsequent mouseovers, bars change color temporarily for emphasis
  • On click, bar changes to red temporarily for emphasis
