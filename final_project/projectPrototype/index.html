Final Project Prototype

This page contains an overview of the prototype for my Information Visualization final project.

Dataset Summary

I am using a dataset of recipes from different countries around the world. The dataset contains a list of ingredients comprising each recipe as well as the country the recipe originated from. My goal with this dataset was to visualize it as a network in order to better understand the kinds of ingredients that exist together in recipes across different regions of the world. The raw data files can be found here: Project Dataset.

Sketch Summary

Technique 1

I have chosen to have the user interact with a world map initially in order to select the region to view recipes from. After clicking on a country, they will be redirected to a bar chart showing the ten most common ingredient pairings for recipes in that country. Since the total number of recipes compiled for each country varies drastically, I have also displayed the number of recipes used in this analysis. Below this chart, a network will be displayed which will portray all ingredients from that country as nodes. Edges will exist between two ingredients if they exist together in a recipe. The edges are also weighted as a functino of how many recipes the two ingredients appear together in. When the user clicks on a different country, a new set of charts will be displayed corresponding to information for the selected country. A more in depth discussion of the sketch can be found on my Project Sketch page.
