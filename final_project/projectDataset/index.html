Final Project Dataset

This page contains information on the dataset I will be visualizing for my final project.

Description

This dataset is a compilation of recipes from around the world. Each row represents a recipe and contains two pieces of information for that recipe - place of origin and list of ingredients. In the original dataset, the place of origin field can be a country, continent, or region of the world (i.e., Eastern Europe). In order to better visualize this data, I have subset it to only contain fields where the place of origin is a country. Below is a rough depiction of what this looks like.

This dataset does not contain the same number of recipe entries per country. Some countries have less than 10 recipes documented, while others have over 40,000. This is because the datset was compiled by scraping three online recipe sites: menupan.com, allrecipes.com, and epicurious.com. While the last two were added by the author in an attempt to compile more foreign recipes, the dataset is still highly skewed to traditional North American cuisine.

Original data location

The original dataset was used as part of a Scientific Reports publication, Flavor Network and the Principles of Food Pairing. It is provided for public use by the author here.

Repository with raw data files

I have chosen to use only a subset of the data used in the original publication. The raw files can be found in my github repository here.

Motivation

My main motivation in using this dataset was the appeal of visualizing a network. Some of my previous internship work dealt with replicating the flavor network built in "Flavor Network and the Principles of Food Pairing". The challenging part of this process was finding an appropriate platform to visualize the network we created. Although I am now working with a different dataset, I would like to explore D3's network implementations and compare them with the network visualizations I had previously built.
